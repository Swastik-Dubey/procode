Dart is a modern general-purpose programming language originally developed by Google (designed by Lars Bak and Kasper Lund).
Dart is a new programming language that appeared first time in 2007 and its latest stable version is released in June 2017.
Not many people knows about Dart, the significant growth of Flutter made it more popular.

Why Dart

The Dart is an open source, object oriented programming language. The Dart provides lots of convenient features mainly focused on compilation, execution, safety, memory management, and concurrency.

Dart features

Object Oriented Programming:-

Dart is an object-oriented programming language with support for inheritance, interfaces and optional typing features

Open Source:-

Dart is an open source programming language and it comes with a set of tools for compiling and executing Dart Code.

Cross Platform:-

Dart is cross-platform and supports all major operating systems.

Browser Support:-

The Dart code can be trans-compiled into JavaScript using source-to-source compilation.

Ahead of times(AOT):-

Dart supports Ahead-Of-Time (AOT) compilation mode. In this mode, the Dart code can be directly converted into native machine code.

Loops

At times, certain instructions require repeated execution. Loops are an ideal way to do the same. A loop represents a set of instructions that must be repeated.
In a loops context, a repetition is termed as an iteration.

The For loop

The for loop is an implementation of a definite loop. The for loop executes the code block for a specified number of times.
It can be used to iterate over a fixed set of values, such as an array

While Loop

The while loop executes the instructions each time the condition specified evaluates to true.
In other words, the loop evaluates the condition before the block of code is executed.

Do-While loop

The do…while loop is similar to the while loop except that the do...while loop doesn’t evaluate the condition for the first time the loop executes.

Break Statement

The break statement is used to take the control out of a construct. Using break in a loop causes the program to exit the loop.
